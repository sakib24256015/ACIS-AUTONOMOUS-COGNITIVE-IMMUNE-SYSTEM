Autonomous Cognitive Immune System for Cyber Defense

ACIS is a Python Flask security operations system with live system telemetry, event logging, authorized web pentest reporting, network monitoring, process intelligence, and printable evidence reports.

REAL TELEMETRY CPU, memory, disk, network interfaces, active connections, and process intelligence from the deployed host.
AUTHORIZED PENTEST Safe web security checks with persistent reports, risk scoring, recommendations, and print-to-PDF support.
AUDIT TRAIL Authentication, scans, findings, report views, and security events are logged in SQLite.
